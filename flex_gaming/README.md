# Flex gaming

## wc

Counts number of lines, number of words and number of characters

Compile:
```
make wc
```

Run:
```
cat wc.l | ./wc
```

## anbn

Matches anbn language. Outputs 0 or 1 if it doesn't or does match the $$a^nb^n$$ language

Compile:
```
make anbn
```

Run:
```
echo "aaabbb" | ./anbn
```
